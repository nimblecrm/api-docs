Pagination metadata
~~~~~~~~~~~~~~~~~~~~

.. code-block:: javascript

    'meta': {
        'per_page': 30,
        'total': 45,
        'pages': 2,
        'page': 1
    }

Keys meaning:

**per_page**
    Number of contacts returned per page
**total**
    Total number of contacts
**pages**
    Total pages count
**page**
    Current page number
